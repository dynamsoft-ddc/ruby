Introduction
The sample demonstrates how to convert image to text by using DDC Ocr Restful Api.

How to run the sample:
1. Update self.getApiKey in configuration.rb with your own secret key. For details on how to get a secret key, please check
   https://developer.dynamsoft.com/authenticating-to-the-cloud-api
2. Run the sample from the command line use the ruby command:
   ruby program.rb
